# YOLO-Annotation-Visualizer
Um visualizador de imagens com anotações no formato YOLO

### Dependencias de libs em python
- cv2
- numpy

### Como rodar
Siga o padrão já definido no exemplo do projeto
- images: pasta onde deve ser colocado as imagens
- labels/yolo: pasta onde deve ser colocado os labels
- classes.names: arquivo com a ordem das classes

Após realizar as alterações necessárias para rodar o seu dataset customizado abra o terminal e digite o código abaixo:
```
python3 visualizeLabels.py
```
